@textbus/core
=====================

Textbus is a rich text framework for building high-performance, highly interactive editors and online documents.

It provides the core document model and editing events. Render the view with [Viewfly](https://viewfly.org), Vue, React, or another adapter.

## Install

```
npm install @textbus/core
```
